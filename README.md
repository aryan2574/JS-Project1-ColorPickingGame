# The GREAT RGB(Red Green Blue) Game!

### A color picking game created using *JavaScript HTML5 CSS*


## In this game RGB value of a color will be shown and you have to guess that color by just clicking on the different colors below. If you chose the correct color it will show Correct otherwise Incorrect.

**In the *Easy* mode you have to guess the correct color among 3 different colors and in the *Hard* mode there wil be 6 different colors.**
